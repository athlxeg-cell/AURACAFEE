Step 5 — Save & Publish

Choose how to save your menu. Option A is easiest. Option B is automatic once set up.

Option A — Download & Upload (No setup required)

1. Click Download menu.json
2. Go to your GitHub repo → click on menu.json → click the ✏️ edit icon → paste the content
3. Commit the change → Netlify auto-deploys in ~30 seconds

Option B — Push Directly to GitHub (Automatic, recommended)

Requires GITHUB_TOKEN, GITHUB_OWNER, GITHUB_REPO set in Netlify environment variables. See the setup guide. Once configured, one click publishes everything.
